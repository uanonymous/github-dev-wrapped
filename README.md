## How it works

This repo is managed automatically by the `github-dev-wrapped` GitHub Action.

Every month it:
1. Collects your GitHub activity via the API
2. Generates an AI-written personality profile with Groq/OpenAI/Gemini
3. Renders a static HTML report and animated SVG card
4. Deploys everything to GitHub Pages

No backend. No database. All data lives in this repo.
